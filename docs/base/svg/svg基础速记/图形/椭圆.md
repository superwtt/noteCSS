### SVG基本图形-椭圆
```javascript
<svg xmlns="http://www.w3.org/2000/svg" version="1.1">
  <ellipse cx="100" cy="100" rx="55" ry="20" fill="red" stroke-width="4" stroke="green"/>
</svg>
```
代码解析：
+ `cx` 属性定义椭圆中心的x坐标
+ `cy` 属性定义椭圆中心的y坐标
+ `rx` 属性定义水平半径
+ `ry` 属性定义垂直半径
