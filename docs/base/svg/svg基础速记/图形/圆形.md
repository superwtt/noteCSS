### SVG基本图形-圆形
```javascript
<svg xmlns="http://www.w3.org/2000/svg" version="1.1">
  <circle cx="100" cy="100" r="40"
   stroke-width="4" stroke="black" fill="red"
  >
</svg>
```
代码解析：
+ cx 和 cy定义圆心的坐标位置
+ r 定义圆的半径
