### SVG基本图形-直线
```javascript
<svg xmlns="http://www.w3.org/2000/svg" version="1.1">
  <line x1="0" y1="0" x2="200" y2="200" stroke-width="3" stroke="red" />
</svg>
```
代码解析：
+ `x1` 属性定义线条在x轴开始的位置
+ `y1` 属性定义线条在y轴开始的位置
+ `x2` 属性定义线条在x轴结束的位置
+ `y2` 属性定义线条在y轴结束的位置
