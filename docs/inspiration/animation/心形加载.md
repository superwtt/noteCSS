### 使用animation实现心形加载
#### 关键点

1. 可以先把最后的心形形状构造出来，方便动画的构建
2. 动画每一帧的设置需要多多尝试

---

#### 代码
```javascript
// 方式一
  //0% {
  //height: 20px;
  //}

  //100% {
  //height: 40px;
  //}

// 方式二
  30%,
  50% {
    height: 60px;
    transform: translateY(-30px);
  }

  75%,
  100% {
    height: 20px;
    transform: translateY(0);
  }
```
方式1 是在4s内完成0%-100%的变化，只能轮流反向播放动画，配合动画延迟动画时间会错乱
方式2 是在4s内完成25%->50%,50%-100%的变化，更加流畅一些
