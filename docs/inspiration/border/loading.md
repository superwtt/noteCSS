### 使用border-style实现各种加载样式

#### 关键点
1. 主要运用border-style的不同取值，配合animation 做出loading的效果
2. border-style 取值有:
   + `solid`
   + `dashed`
   + `dotted`
   + `double`
   + `groove`
   + `ridge`
   + `inset`
   + `outset`
