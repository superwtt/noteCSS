### CSS实现瀑布流布局(column-count)

瀑布流布局常见于商品的陈列展示。本例使用CSS `column-count`属性实现

---

### 原理
+ `column-count`：规定元素应该被划分的列数
+ `Column-gap`：规定列元素之间的距离
+ `break-inside`：避免在元素内部插入分页符
